# Precedent Study Datasheet

**Author:** Yizhang Mu (he/him)

## Project Overview

This website presents a precedent study of **The Evros/Meriç River: A Century of Border Design** by Forensic Architecture. It transforms a conventional research datasheet into an interactive digital archive inspired by archaeological documents and investigation notebooks.

---

## Aesthetic Style

The visual language is inspired by:

- Archaeological archives
- Vintage research notebooks
- Museum catalogues
- Handcrafted paper textures
- Forensic investigation documents

The interface uses warm cream and ochre colors, paper textures, masking tape, handwritten notes, and soft shadows to create the feeling of exploring historical evidence rather than browsing a standard website.

---

## Website Structure

The website consists of three pages:

### 01 Overview
- Project summary
- Basic precedent information
- Author
- Format
- Intended audience

### 02 Studio & Team Information
- About Forensic Architecture
- Core Team
- Studio timeline
- Author bibliography

### 03 Sources & References
- Research focus
- Reference bibliography
- Investigation methods
- Research tools

---

## Technologies

- HTML5
- CSS3
- Responsive Grid Layout
- Flexbox
- Google Fonts

---

## Design Goals

The goal is to create a digital archive that feels tactile, calm, and immersive while maintaining readability. Instead of a conventional webpage, the interface encourages users to experience research as if turning the pages of an investigation notebook.
